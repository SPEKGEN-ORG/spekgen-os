Para todo contenido de @gibran.alonzo.ecom: tono de **amigo compartiendo tips**, no de guru/experto/autoridad agresiva.

**Why:** Gibran lo dijo explícito 2026-05-10 después de revisar un brief de contenido para GAE: "no me gusta el framing de 'mira como yo hago esto y tu no'... QUIERO POSTS MÁS ÚTILES PARA MI AUDIENCIA, MÁS ORGÁNICOS, QUE SE SIENTAN COMO UN AMIGO COMPARTIENDO SUS TIPS". Él tiene un ICP que está empezando — un tono superior los aleja, un tono útil los convierte en seguidores. El posicionamiento como experto se gana entregando valor consistente, NO afirmándolo.

**How to apply:** Aplica a todos los formatos de GAE (carruseles, reels, captions, stories) y a todo análisis/brief de contenido que produzca para Gibran.

### NUNCA hacer

1. **NUNCA mencionar # de clientes que tiene** ("5 clientes activos", "manejo X cuentas") — regla preexistente que ya había violado. Ningún número de clientes, empleados, ingresos, ni "agencia 99.99% AI" en contenido público. Esos son datos internos.
2. **NUNCA framing de "mira cómo yo hago esto y tu no"** — superioridad implícita o explícita.
3. **NUNCA "lo estás haciendo mal"** como hook — es agresivo, ahuyenta al ICP que está empezando.
4. **NUNCA autobombo de la thesis** ("yo demuestro que se puede tener una agencia 99.99% AI") — eso es para conversaciones privadas, no para contenido orgánico.
5. **NUNCA posturing aspiracional vacío** ("el que entiende este shift gana la próxima década") — suena a guru de LinkedIn.

### SÍ hacer

1. **Hook desde un dolor concreto que el viewer ya tiene.** Ejemplos que Gibran propuso él mismo y aprobó:
   - "Tu Claude pierde el hilo a la mitad del proyecto? Usa /compact"
   - "Si tu sesión de Claude se siente cara, casi siempre es exceso de contexto"
2. **Tactical, accionable, ejecutable.** "3 comandos que uso todos los días para ahorrar tokens" >>> "los principios de cómo opero".
3. **Hot takes útiles.** "No uses Claude sin la feature de multi-agentes" — opinión fuerte respaldada por valor.
4. **Tono primera persona casual.** "Yo uso esto", "mi setup actual", "me funciona porque...". NO "demuestro que", "comparte mi metodología".
5. **CTA suave** que invite participación, no que demande seguir. "Cuál usas más?", "Te ha pasado?".

### ICP recordatorio
Dueños $80K-500K MXN/mes que están empezando a usar Claude Code o quieren empezar. Posicionamiento de Gibran: experto pragmático que comparte lo que aprende. NO guru, NO influencer "AI agency", NO LinkedIn-philosopher.

### Formato de briefs
Cuando produzca briefs de contenido para GAE: **mínimo 3 ideas por formato** (3 carruseles + 3 reels), no 1 sola idea pulida. Gibran cura desde un menú, no aprueba/rechaza una sola opción.
