## Rule

Every piece of copy for any client (captions, headlines, CTAs, notes) MUST use correct Spanish orthography:
- ñ (años, daño, mañana, señal)
- tildes/acentos (articulación, inflamación, también, más, además, después, crónico, médico, cartílago, específico)
- ¿? and ¡! when applicable

## Critical Words (never write without accent)

| Correct | WRONG (never use) |
|---|---|
| años | anos |
| daño | dano |
| articulación | articulacion |
| inflamación | inflamacion |
| también | tambien |
| más | mas (as adverb) |
| crónico | cronico |
| médico | medico |
| cartílago | cartilago |
| tú | tu (when "you") |
| instalé | instale |
| probé | probe |
| gustó | gusto |
| cerré | cerre |
| seguí | segui |
| después | despues |
| entendí | entendi |
| área | area |
| creación | creacion |
| sígueme | sigueme |
| cuál | cual (interrogative) |
| cuáles | cuales (interrogative) |
| qué | que (interrogative) |
| cómo | como (interrogative) |
| quién | quien (interrogative) |
| próximo | proximo |

## Verification Before Delivery

Before delivering any copy, search for "anos", "dano", "articulacion", "inflamacion", "tambien", "tu " (sin acento), "despues", "proximo", "cual " — if any appear without accents, fix them.
